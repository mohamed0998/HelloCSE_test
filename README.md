## :hugs: Requirements

        *  Laravel 9
        *  Composer
        *  Vuejs 3

## :clap: Installation:

```bash
    composer install

    npm install

    cp .env.example .env
    DB_DATABASE=test
    # Database Connection on .env
    
    php artisan migrate:fresh

    # Run Backend Server
    php artisan serve

    # Run Frontend Server
    npm run watch

```
